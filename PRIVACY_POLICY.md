# Политика конфиденциальности приложения "Антимошенник"

**Дата вступления в силу:** 18 декабря 2024 г.

## 1. Общие положения

Приложение "Антимошенник" (далее — "Приложение") предназначено для защиты пользователей от телефонного мошенничества путём анализа записей телефонных разговоров.

## 2. Сбор и обработка данных

### 2.1. Какие данные обрабатываются:
- **Аудиозаписи звонков** — анализируются локально на устройстве для выявления признаков мошенничества
- **Номера телефонов** — только те, которые пользователь добавляет в белый список или указывает как SOS-контакт

### 2.2. Как обрабатываются данные:
- Все данные обрабатываются **исключительно на устройстве пользователя**
- Приложение **НЕ передаёт** аудиозаписи или персональные данные на внешние серверы
- Распознавание речи происходит офлайн с использованием технологии Vosk

## 3. Разрешения приложения

Приложение запрашивает следующие разрешения:

| Разрешение | Назначение |
|------------|------------|
| Доступ к файлам | Чтение записей звонков для анализа |
| Телефонные звонки | Совершение SOS-звонка родственнику |
| Уведомления | Показ предупреждений об угрозах |

## 4. Хранение данных

- История анализа хранится локально на устройстве
- Пользователь может удалить историю в любой момент через настройки приложения
- При удалении приложения все данные удаляются

## 5. Передача данных третьим лицам

Приложение **НЕ передаёт** никакие данные третьим лицам, рекламным сетям или аналитическим сервисам.

## 6. Безопасность

Приложение работает полностью автономно и не требует подключения к интернету для основных функций анализа.

## 7. Изменения политики

Мы можем обновлять данную политику. Актуальная версия всегда доступна в приложении и на странице в магазине.
